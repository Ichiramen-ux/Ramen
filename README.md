# 🍜 Ramen Bot

A Discord bot that spawns and eats digital ramen!

## Commands

- `!ramen` - Spawn a ramen bowl
- `!eat` - Eat the ramen with animation
- `!stats` - View your ramen stats

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your `DISCORD_TOKEN`
3. Run: `python main.py`
